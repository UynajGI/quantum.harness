Exact diagonalization — expert interview

An interview script for talking to a working exact diagonalization (ED) practitioner. Cues are prompts, not answer choices. The useful answer is usually: what signal did you notice, what did it mean, and what did you do next?

Interview rhythm. Ask the question, let the expert answer freely, then use cues only if the answer stalls. Tick cues the expert touched on. When an answer is shaped like a lesson, capture it as signal (what they noticed) → understanding (what it meant) → action (what to do), plus scope (where the lesson applies) and exception (where it breaks).
